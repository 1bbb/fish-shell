\section listeners listeners - find which functions are receiving event, signal, variable change, and exit notifications.

\subsection function-synopsis Synopsis
<code>listeners [OPTIONS]</code>

\subsection listeners-description Description

This builtin command is used to show which functions are registered to receive events, signals, and other notifications.

- <code>-e EVENTNAME</code> or <code>--event=EVENTNAME</code> show functions that will be notified when the given event is triggered.
- <code>-s SIG</code> or <code>--signal=SIG</code> show functions that will be notified when the given signal is received.
- <code>-v NAME</code> or <code>--variable=NAME</code> show functions that will be notified when the given variable is altered. 
- <code>-p PID</code> or <code>--process-exit=PID</code> show functions that will be notified when the given process exits.
- <code>-j JOBID</code> or <code>--job-exit=JOBID</code> show functions that will be notified when the given job exits.
- <code>-h</code> or <code>--help</code> display a help message and exit
